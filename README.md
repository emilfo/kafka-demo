Git repo som reflekterer koden som ble produsert i foredrag på Knowits JVM-summit 3
https://youtu.be/XqMLz_v7xas?t=4075

## Konsepter som blir gått over

SelectKey
Ktable
GroupBy
Count (reduce)
Aggregate
Join
FlatMap
